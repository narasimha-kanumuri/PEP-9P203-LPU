# 📌 Problem Statement

- You are given a binary matrix of size N × M.

- A square submatrix is called **alternating** if:
- - Adjacent cells (up, down, left, right) always have different values
- - The pattern follows a **consistent chessboard structure**

- This means:
- - For any cell (i, j), its value must follow:
  
$$ grid[i][j] = (i + j) \bmod 2 \quad \text{OR its inverse} $$

---

## 🎯 Objective

- Find the **side length** of the largest alternating square submatrix.

---

## 📥 Input Format

- - First line: two integers N and M  
- - Next N lines: M binary integers (0 or 1)

---

## 📤 Output Format

- - A single integer → maximum side length of valid alternating square

---

## 🧠 Constraints

- - 1 ≤ N, M ≤ 1000  
- - grid[i][j] ∈ {0, 1}

---

## ⚠️ Important Clarification

- - Alternation is NOT local-only
- - Entire square must match one of:
-   - Standard chessboard pattern
-   - Inverted chessboard pattern

---

## 🧪 Sample Inputs & Outputs

### Sample 1

```text
1 0 1
0 1 0
1 0 1
```

- **Output:**
- 3

---

### Sample 2

```text
1 1 0
0 1 0
1 0 1
```

- **Output:**
- 2

---

## 🤔 Edge-Case Thinking Prompts

- - What if all values are same?
- - What if matrix is 1 × 1?
- - Can a valid square start anywhere?
- - Does pattern depend on absolute indices or relative position?

---

## 🧠 Thinking Like a Programmer

### What must be tracked?
- - Size of valid alternating square ending at each cell

### What can change?
- - Pattern alignment (normal vs inverted)

### What must never change?
- - Chessboard alternation rule

### What brute force would do?
- - Check every square → `O(N³)`
