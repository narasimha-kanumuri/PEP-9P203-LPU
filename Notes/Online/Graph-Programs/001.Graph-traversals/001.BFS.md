# BFS example states of a graph traversal

**Graph representation:** 5 nodes, edges: (0-1), (0-2), (1-3), (2-3), (3-4)

**BFS traversal order starting from node 0:** 0 → 1 → 2 → 3 → 4

**BFS queue states during traversal:**

1. Start: Queue = [0], Visited = {0}
2. Dequeue 0: Queue = [], Visited = {0}
3. Enqueue neighbors of 0: Queue = [1, 2], Visited = {0, 1, 2}
4. Dequeue 1: Queue = [2], Visited = {0, 1, 2}
5. Enqueue neighbors of 1: Queue = [2, 3], Visited = {0, 1, 2, 3}
6. Dequeue 2: Queue = [3], Visited = {0, 1, 2, 3}
7. Enqueue neighbors of 2: Queue = [3], Visited = {0, 1, 2, 3} (3 already visited)
8. Dequeue 3: Queue = [], Visited = {0, 1, 2, 3}
9. Enqueue neighbors of 3: Queue = [4], Visited = {0, 1, 2, 3, 4}
10. Dequeue 4: Queue = [], Visited = {0, 1, 2, 3, 4}
