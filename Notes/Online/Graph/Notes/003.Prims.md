# Prim's Algorithm 

Prim's algorithm is a greedy algorithm used to find the minimum spanning tree (MST) of a weighted undirected graph. The algorithm starts with an arbitrary node and grows the MST by repeatedly adding the smallest edge that connects a vertex in the MST to a vertex outside the MST.

## Minimum Spanning Tree (MST)

**Definition:** A minimum spanning tree of a graph is a subset of the edges that connects all the vertices together, without any cycles and with the minimum possible total edge weight.

## Prim's Algorithm Steps

1. Initialize a tree with a single vertex, chosen arbitrarily from the graph.
2. While the tree does not contain all the vertices:
   - Find the edge with the smallest weight that connects a vertex in the tree to a vertex outside the tree.
   - Add that edge and the vertex it connects to the tree.
   - Repeat until all vertices are included in the tree.
   - The algorithm can be implemented using a priority queue (min-heap) to efficiently retrieve the edge with the smallest weight at each step.
  
```cpp

#include <iostream>
#include <vector>
#include <queue>
#include <utility>
#include <limits>

using namespace std;

vector<pair<int, int>> primMST(int n, vector<vector<pair<int, int>>>& graph) {
    vector<bool> inMST(n, false);
    vector<pair<int, int>> mstEdges; // To store the edges of the MST
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> minHeap; // Min-heap for edges

    // Start from the first vertex (index 0)
    minHeap.push({0, 0}); // {weight, vertex}

    while (!minHeap.empty()) {
        auto [weight, vertex] = minHeap.top();
        minHeap.pop();

        if (inMST[vertex]) continue; // Skip if vertex is already in MST

        inMST[vertex] = true; // Mark the vertex as included in MST

        // If it's not the starting vertex, add the edge to the MST
        if (weight != 0) {
            mstEdges.push_back({weight, vertex});
        }

        // Add all adjacent edges to the min-heap
        for (auto& [adjWeight, adjVertex] : graph[vertex]) {
            if (!inMST[adjVertex]) {
                minHeap.push({adjWeight, adjVertex});
            }
        }
    }

    return mstEdges;
}
```
