# List of all keywords and terminologies of graph concepts:

**List of keywords , terminologies in learning order, with small definitions, and examples in table format:**

| Keyword/Terminology | Definition | Example |
|---------------------|------------|---------|
| Vertex (Node) | A fundamental unit of a graph, representing an entity. | In a social network graph, a vertex could represent a person. |
| Edge | A connection between two vertices in a graph. | In a social network graph, an edge could represent a friendship between two people. |
| Directed Graph | A graph where edges have a direction, indicating a one-way relationship. | In a Twitter follower graph, an edge from user A to user B indicates that A follows B, but not necessarily the other way around. |
| Undirected Graph | A graph where edges do not have a direction, indicating a two-way relationship. | In a Facebook friendship graph, an edge between user A and user B indicates that they are friends, which is a mutual relationship. |
| Weighted Graph | A graph where edges have weights or costs associated with them. | In a road network graph, the weight of an edge could represent the distance or travel time between two locations. |
| Unweighted Graph | A graph where edges do not have weights or costs associated with them. | In a social network graph, the edges may simply represent connections between people without any additional information. |
| Path | A sequence of vertices connected by edges in a graph. | In a transportation network graph, a path could represent a route from one city to another through intermediate cities. |
| Cycle | A path that starts and ends at the same vertex without repeating any edges. | In a graph representing a computer network, a cycle could represent a loop in the network where data can circulate indefinitely. |
| Degree | The number of edges connected to a vertex. | In a social network graph, the degree of a vertex (person) could represent the number of friends they have. |
| Connected Graph | A graph where there is a path between every pair of vertices. | In a transportation network graph, a connected graph would mean that you can travel from any city to any other city through the network. |
| Disconnected Graph | A graph where there are at least two vertices that do not have a path between them. | In a social network graph, a disconnected graph could represent two groups of people who have no connections between them. |
| Subgraph | A graph formed from a subset of the vertices and edges of a larger graph. | In a social network graph, a subgraph could represent a specific community or group of friends within the larger network. |
