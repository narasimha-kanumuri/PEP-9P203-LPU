# 14 Maximum Subarray Sum with One Modification

## Detailed Problem Statement

Given an array of integers `nums`, find the maximum subarray sum you can achieve by modifying at most one element in the array. You can modify an element to **any value** you choose. The subarray must be contiguous, and the modification can be applied to any element in the array, including the first and last elements. The element you modify can be part of the subarray or outside of it. The rule to pick the replaced value is that it can be any integer, including negative values. 

## Constraints

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

## Examples

### Example 1

**Input**: nums = [1, -2, 3, 4]
**Output**: 10

**Explanation**: By modifying the element -2 to 5, we can get the subarray [1, 5, 3, 4] which has a sum of 10.

### Example 2

**Input**: nums = [-1, -2, -3]
**Output**: 0

**Explanation**: By modifying the element -1 to 0, we can get the subarray [0, -2, -3] which has a sum of 0.

### Example 3

**Input**: nums = [2, -1, 2]
**Output**: 7

**Explanation**: By modifying the element -1 to 2, we can get the subarray [2, 2, 2] which has a sum of 6. However, the maximum subarray sum we can achieve is by modifying the element -1 to 3, which gives us the subarray [2, 3, 2] with a sum of 7.

## ELI5 Explanation

Imagine you have a row of boxes, each with some candies in them. You want to pick a group of consecutive boxes to get the most candies. Normally, you just add up the candies in the boxes you pick. But now, you have a magic power: you can change the number of candies in one box to any number you want, even a negative number. Your goal is to use this magic power wisely to get the maximum total candies from a group of consecutive boxes.

## Implementation Narrative

To solve this problem, we can use a dynamic programming approach similar to Kadane's algorithm for finding the maximum subarray sum. However, we need to account for the possibility of modifying one element. We can maintain two arrays: `dp_no_mod` and `dp_mod`, where `dp_no_mod[i]` represents the maximum subarray sum ending at index `i` without any modification, and `dp_mod[i]` represents the maximum subarray sum ending at index `i` with one modification.

1. We will iterate through the array and fill these two arrays based on the following rules:
   - For `dp_no_mod[i]`, we can either start a new subarray at index `i` or extend the previous subarray without modification.
   - For `dp_mod[i]`, we can either modify the current element and start a new subarray, or extend the previous modified subarray, or extend the previous non-modified subarray by modifying the current element.
   - We will keep track of the maximum values in both `dp_no_mod` and `dp_mod` to find the overall maximum subarray sum.
   - Finally, we will return the maximum of the two arrays as our answer.
2. The time complexity of this approach is O(n) and the space complexity is O(n) due to the two arrays we maintain.

## C++ Implementation

```cpp

#include <vector>
#include <algorithm>
#include <iostream>
using namespace std;

class Solution {
public:
int maxSubarraySumWithOneModification(vector<int>& nums) {// Function to find the maximum subarray sum with at most one modification
        int n = nums.size();// Get the size of the input array
        vector<int> dp_no_mod(n, 0);// dp_no_mod[i] will store the maximum subarray sum ending at index i without modification
        vector<int> dp_mod(n, 0);// dp_mod[i] will store the maximum subarray sum ending at index i with one modification

        dp_no_mod[0] = nums[0];// Initialize the first element of dp_no_mod with the first element of nums
        dp_mod[0] = max(nums[0], 0); // Modify the first element to 0 if it's negative

        int max_sum = max(dp_no_mod[0], dp_mod[0]);// Initialize max_sum with the maximum of the first elements of both dp arrays

        for (int i = 1; i < n; i++) {// Iterate through the array starting from the second element
            dp_no_mod[i] = max(nums[i], dp_no_mod[i - 1] + nums[i]);// Update dp_no_mod for the current index
            dp_mod[i] = max({dp_mod[i - 1] + nums[i], dp_no_mod[i - 1] + max(nums[i], 0), max(nums[i], 0)});// Update dp_mod for the current index considering all three cases
            max_sum = max({max_sum, dp_no_mod[i], dp_mod[i]});// Update max_sum with the maximum of the current elements of both dp arrays
        }

        return max_sum;
    }
};

// Example usage
int main() {
    Solution sol;
    vector<int> nums = {1, -2, 3, 4};
    cout << sol.maxSubarraySumWithOneModification(nums) << endl; // Output: 10
    return 0;
}
```
