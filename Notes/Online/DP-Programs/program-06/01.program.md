## Problem Statement

Given a 2D matrix of integers, find the size of the largest square submatrix where the values alternate in a chessboard pattern (i.e., no two adjacent cells have the same value). The size of the square is defined as the length of its side.
**Twist: This problem can be solved using dynamic programming by maintaining a state that checks for the alternating pattern while building up the solution.**

## Input Format

- The first line contains two integers `n` and `m`, the number of rows and columns in the matrix, respectively.
- The next `n` lines each contain `m` space-separated integers representing the values in the matrix.
- The values in the matrix can be any integer, and the alternating pattern is determined by whether adjacent cells (horizontally and vertically) have different values.

## Output Format

- Output a single integer, the size of the largest square submatrix that follows the alternating pattern. If no such square exists, output `0`.
- If the matrix is empty (i.e., `n` or `m` is `0`), output `0`.
- If all values in the matrix are the same, output `1` (since a single cell is considered a valid alternating square).
- If the largest alternating square is of size `1`, output `1`.
- If the largest alternating square is of size greater than `1`, output the size of that square.
- If there are multiple largest alternating squares, output the size of any one of them.
- If the input matrix contains negative values, they should be treated as valid integers for the alternating pattern.
- If the input matrix contains zero values, they should be treated as valid integers for the alternating pattern.
- If the input matrix contains a mix of positive, negative, and zero values, the alternating
- pattern should still be determined based on whether adjacent cells have different values, regardless of their sign.

In the given matrix, the largest square submatrix that follows the alternating pattern is of size `4x4`, which can be found in the top-left corner of the matrix. The values alternate between `1` and `2` in a chessboard pattern, making it a valid alternating square. Therefore, the output is `4`.

## Algorith in-depth explanation of the solution:

To solve the problem of finding the largest alternating square in a matrix, we can use dynamic programming. The idea is to create a DP table where `dp[i][j]` represents the size of the largest alternating square that can be formed with the bottom-right corner at cell `(i, j)`.

The algorithm can be broken down into the following steps:

1. **Initialization**: Create a DP table of the same size as the input matrix, initialized to `1` for all cells, since the smallest alternating square is a single cell.
2. **Filling the DP Table**: Iterate through the matrix starting from the second row and second column. For each cell `(i, j)`, check if it forms an alternating pattern with its top `(i-1, j)`, left `(i, j-1)`, and top-left `(i-1, j-1)` neighbors. If it does, update `dp[i][j]` to be the minimum of the three neighboring DP values plus one.
3. **Tracking the Maximum Size**: Keep track of the maximum value in the DP table, which will represent the size of the largest alternating square found.
4. **Return the Result**: After filling the DP table, return the maximum size found.
5. **Edge Cases**: Handle edge cases such as empty matrices, matrices with all identical values, and matrices with negative or zero values appropriately.

#include <iostream>
#include <vector>
using namespace std;

int largestAlternatingSquare(vector<vector<int>>& matrix) {
    int n = matrix.size();
    if (n == 0) return 0;
    int m = matrix[0].size();
    if (m == 0) return 0;

    vector<vector<int>> dp(n, vector<int>(m, 1));
    int maxSize = 1;

    for (int i = 1; i < n; i++) {
        for (int j = 1; j < m; j++) {
            if (matrix[i][j] != matrix[i-1][j] && 
                matrix[i][j] != matrix[i][j-1] && 
                matrix[i-1][j] != matrix[i][j-1]) {
                dp[i][j] = min({dp[i-1][j], dp[i][j-1], dp[i-1][j-1]}) + 1;
                maxSize = max(maxSize, dp[i][j]);
            }
        }
    }

    return maxSize;
}

int main() {
    int n, m;
    cin >> n >> m;
    vector<vector<int>> matrix(n, vector<int>(m));
    
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            cin >> matrix[i][j];
        }
    }

    cout << largestAlternatingSquare(matrix) << endl;
    return 0;
}
