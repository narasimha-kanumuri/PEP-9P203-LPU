# 13️⃣ Longest Palindromic Subsequence with One Deletion

## Problem Statement

Given a string `s`, find the length of the longest palindromic subsequence in `s` after deleting at most one character from it.
A subsequence is a sequence that can be derived from another sequence by deleting some or no elements without changing the order of the remaining elements. A palindrome is a string that reads the same backward as forward. 

## Constraints

- `1 <= s.length <= 1000`
- `s` consists of lowercase English letters.
- You can delete at most one character from the string.
- The length of the longest palindromic subsequence should be returned as an integer.

## Examples

### Example 1

**Input**: s = "abca"
**Output**: 3

**Explanation**: By deleting the character 'b', we can get the longest palindromic subsequence "aca", which has a length of 3.

### Example 2

**Input**: s = "abc"
**Output**: 1

**Explanation**: By deleting any character, we can only get a single character as a palindrome, which has a length of 1.

### Example 3

**Input**: s = "aebcbda"
**Output**: 3
**Explanation**: By deleting the character 'e', we have the string "abcbda" which is not a palindrome.  By deleting the character 'd', we have the string "aebcba" which is not a palindrome. Therefore, the longest palindrome is ONLY "bcb", which has a length of 3. There are no more palindromic subsequences that can be formed by deleting one character. Therefore, the longest palindromic subsequence is "bcb" with a length of 3.

## ELI5 Explanation

Imagine you have a string of letters, and you want to find the longest sequence of letters that reads the same backward and forward (a palindrome). However, you are allowed to remove one letter from the string if it helps you get a longer palindrome. The task is to figure out how long that longest palindrome can be after you remove at most one letter.

## Implementation Narrative

To solve this problem, we can use dynamic programming to find the longest palindromic subsequence in the string. We will create a 2D array `dp` where `dp[i][j]` represents the length of the longest palindromic subsequence in the substring `s[i...j]`.

1. We will initialize the base cases where `dp[i][i] = 1` for all `i`, since a single character is always a palindrome of length 1.
2. We will fill the `dp` table in a bottom-up manner, checking if the characters at the current indices are the same. If they are, we can extend the palindrome by 2 (i.e., `dp[i][j] = dp[i+1][j-1] + 2`). If they are not, we take the maximum of the two possibilities (i.e., `dp[i][j] = max(dp[i+1][j], dp[i][j-1])`).
3. After filling the `dp` table, the length of the longest palindromic subsequence with at most one deletion will be the maximum value in the `dp` table.
4. To account for the one deletion, we will iterate through the string and try deleting each character, then calculate the longest palindromic subsequence for the resulting string using the `dp` table.

## Trace table for the string "abca":

|   | a | b | c | a |
|---|---|---|---|---|
| a | 1 | 1 | 1 | 3 |
| b |   | 1 | 1 | 1 |
| c |   |   | 1 | 1 |
| a |   |   |   | 1 |

In this table, `dp[0][3]` gives us the length of the longest palindromic subsequence in the entire string "abca", which is 3 (the subsequence "aca"). By deleting 'b', we can achieve this longest palindrome.

## Cpp Implementation

```cpp

#include <iostream>
#include <vector>
#include <string>
using namespace std;

class Solution {
public:
    int longestPalindromeSubseq(string s) {
        int n = s.size();
        vector<vector<int>> dp(n, vector<int>(n, 0));

        for (int i = n - 1; i >= 0; i--) {
            dp[i][i] = 1; // Base case: single character is a palindrome
            for (int j = i + 1; j < n; j++) {// Fill the dp table
                if (s[i] == s[j]) { // If characters match, extend the palindrome
                // If characters match, extend the palindrome by 2 and add the length of the longest palindromic subsequence in the substring between them
                    dp[i][j] = dp[i + 1][j - 1] + 2; // If characters match, extend the palindrome by 2 and add the length of the longest palindromic subsequence in the substring between them
                } else { // If characters do not match, take the maximum of the two possibilities (i.e., excluding one character from either end)
                    dp[i][j] = max(dp[i + 1][j], dp[i][j - 1]); // If characters do not match, take the maximum of the two possibilities (i.e., excluding one character from either end)
                }
            }
        }

        int maxLength = dp[0][n - 1]; // Longest palindromic subsequence without deletion

        // Check for one deletion
        for (int i = 0; i < n; i++) {
            // Try deleting character at index i
            int leftLength = (i > 0) ? dp[0][i - 1] : 0;
            int rightLength = (i < n - 1) ? dp[i + 1][n - 1] : 0;
            maxLength = max(maxLength, leftLength + rightLength);
        }

        return maxLength;
    }
};

int main() {
    Solution sol;
    string s = "abca";
    cout << sol.longestPalindromeSubseq(s) << endl; // Output: 3
    return 0;
}
```
