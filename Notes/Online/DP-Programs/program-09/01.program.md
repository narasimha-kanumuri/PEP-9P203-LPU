## Problem Statement

Given a string `s` and a dictionary of words `wordDict`, determine the minimum number of cuts needed to segment the string into a space-separated sequence of one or more dictionary words. If it is not possible to segment the string using the dictionary, return -1.
**Twist: This problem can be solved using dynamic programming with optimization techniques such as prefix checking to efficiently determine if a substring can be segmented.**

## Input Format

- The first line contains a string `s`, the input string to be segmented.
- The second line contains an integer `n`, the number of words in the dictionary.
- The next `n` lines each contain a single word, representing the words in the dictionary.
- The input string `s` will consist of lowercase English letters only, and the dictionary will also contain lowercase English words.
- The length of the input string `s` will be between 1 and 1000, and the number of words in the dictionary will be between 1 and 1000. Each word in the dictionary will have a length between 1 and 20.

## Output Format

- Output a single integer, the minimum number of cuts needed to segment the string into valid dictionary words. If it is not possible to segment the string, output -1.
- The number of cuts is defined as the number of spaces needed to separate the segmented words. For example, if the segmented string is "apple pen", it requires 1 cut (space) to separate "apple" and "pen". If the segmented string is "apple", it requires 0 cuts since it is already a valid word without needing any separation.
- The solution should efficiently compute the minimum cuts by utilizing dynamic programming to store intermediate results and optimize the checking of valid segments using the dictionary.

## Constraints

- 1 ≤ length of s ≤ 1000
- 1 ≤ n ≤ 1000
- 1 ≤ length of each word in the dictionary ≤ 20
- The input string `s` and the words in the dictionary will consist of lowercase English letters only.
- The output will be a single integer representing the minimum number of cuts needed to segment the string, or -1 if segmentation is not possible.
- The solution should be optimized to handle the worst-case scenario within the given constraints, ensuring that it runs efficiently for longer input strings and larger dictionaries.

## Step by step English to Algorithm of the problem

1. Create a boolean array `dp` of size `len(s) + 1` to store whether the substring up to index `i` can be segmented.
2. Initialize `dp[0]` to `True` since an empty string can be segmented.
3. Create an integer array `cuts` of size `len(s) + 1` to store the minimum cuts needed for the substring up to index `i`.
4. Initialize `cuts[0]` to -1 since no cuts are needed for an empty string.
5. Iterate through the string from index 1 to `len(s)`.
   - For each index `i`, iterate through the dictionary of words.
   - For each word in the dictionary, check if the substring from index `i - len(word)` to `i` matches the word and if `dp[i - len(word)]` is `True`.
   - If both conditions are satisfied, update `dp[i]` to `True` and update `cuts[i]` to the minimum of its current value and `cuts[i - len(word)] + 1`.
   - Continue this process until all indices have been processed.
6. After processing, if `dp[len(s)]` is `True`, return `cuts[len(s)]`, otherwise return -1.


## Cpp

```cpp

#include <iostream>
#include <vector>
#include <unordered_set>
#include <string>
using namespace std;

// Function to determine the minimum number of cuts needed to segment the string
int minCut(string s, vector<string>& wordDict) {
    // Create a set for quick lookup of dictionary words
    unordered_set<string> dict(wordDict.begin(), wordDict.end());
    int n = s.size();// Length of the input string
    vector<bool> dp(n + 1, false);// DP array to store whether the substring up to index i can be segmented
    vector<int> cuts(n + 1, INT_MAX);// Array to store the minimum cuts needed for the substring up to index i
    
    dp[0] = true; // Base case: empty string can be segmented
    cuts[0] = -1; // No cuts needed for an empty string
    
    // Iterate through the string from index 1 to n
    for (int i = 1; i <= n; ++i) {
        // Check each word in the dictionary
        for (int j = 0; j < i; ++j) {
            // Check if the substring from index j to i matches a word in the dictionary and if dp[j] is true
            if (dp[j] && dict.count(s.substr(j, i - j))) {
                dp[i] = true;
// Update cuts[i] to the minimum of its current value and cuts[j] + 1
                cuts[i] = min(cuts[i], cuts[j] + 1);
            }
        }
    }
    
    return dp[n] ? cuts[n] : -1;
}

int main() {
    string s;
    int n;
    cin >> s >> n;
    vector<string> wordDict(n);
    for (int i = 0; i < n; ++i) {
        cin >> wordDict[i];
    }
    
    int result = minCut(s, wordDict);
    cout << result << endl;
    
    return 0;
}
```
