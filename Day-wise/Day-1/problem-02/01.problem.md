# 2Ô∏è‚É£ Problem Story / Context

You are monitoring a stream of integer readings coming from a sensor.

Due to signal noise, the same value may reappear. However, if a value repeats too soon (within **K readings**), it indicates a malfunction.

Your task is to detect whether **any value appears at least twice** such that the difference between their indices is **less than or equal to K**.

Return `true` if such a pair exists. Otherwise, return `false`.
